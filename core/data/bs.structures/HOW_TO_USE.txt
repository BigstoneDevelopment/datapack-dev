This is how to use the rewritten api for item structures

Save structure:
/function bs.structures:save {"x": 0, "y": 0, "z": 0}

Load Structure:
/function bs.structures:load {"x": 0, "y": 0, "z": 0}


Import structure:
/data modify storage bigstone_sandbox:structures structureData ..

Export Structure:
/data get storage bigstone_sandbox:structures save ..


Init Library:
/function bs.structures:zprivate/init

Uninstall Library:
/function bs.structures:uninstall

Cancel current function:
/function bs.structures:cancel
